Imagine an online game where users can compete with their scores. However, to make it more challenging, the game takes two sets of scores instead of one. Your task is to write a program that initially takes two integers n, m (1 <= n, m <= 50) representing the number of scores in each set respectively. Then, it takes n positive integers representing the score set A, and m positive integers representing the score set B. The program should find the number of common scores in both sets. Each score is less than or equal to 100.


Sample Input:
4 5
2 4 6 2
1 4 2 1 4

Sample Output:
2


Sample Input:
3 3
3 7 5
2 5 8

Sample Output:
1



Memory limit: 256 MB
Time limit: 5 seconds
